## * Завдання *8*
Оглянуто роботу приватних атрибутів та методів у Python.
Продемонстровано різницю між захищеними (_) та приватними (__) атрибутами.
Перевірено доступ до них напряму, через проперті та через name-mangling.    

![1](/4_lab/pic/8.png)
## * Завдання *9*   
Оглянуто магічні методи ```__repr__() та __str__().```
Перевірено, як саме Python виводить об’єкт у різних контекстах та яку інформацію повертає кожен із методів.     
![1](/4_lab/pic/9.png)
## * Завдання *10*      
Оглянуто арифметичні та службові магічні методи, такі як ```__sub__, __gt__, __len__.```
Перевірено роботу операторів віднімання, порівняння та визначення довжини, а також їх взаємодію з об’єктами класу.
