   1. Програма вивела значення: 
```python
print("1. Обробка винятків 'try...except...finally':")
A = 0
try:
    print("Що буде якщо 10/A ?")
    result = 10 / A  # Спровокуємо ZeroDivisionError
    print("Результат: ", result)
except ZeroDivisionError as e:
    # Виловлюємо конкретну помилку
    print("Невже це помилка ділення на нуль > ", e)
except Exception as e:
    # Виловлюємо інші неочікувані помилки
    print("Невідома помилка > ", e)
finally:
    # Блок виконується завжди

```

```python

print("1. Контекст-менеджер 'with' для читання файлу:")

    with open("test_file.txt", "r") as f:
        # Конструкція with гарантує виклик f.close() після виходу з блоку
```

### Результати виконання завдання 8 (Lambda-функції);

    1. Розробили/Створили: Код, що демонструє **lambda-функцію** для множення та як **ключ сортування (key)**.

    1. Програма вивела значення: 
```python
# Приклад 1: Виклик звичайної функції
def multiply_func(x, y):
    return x * y

# Приклад 2: Виклик lambda-функції
multiply_lambda = lambda x, y: x * y

print("1. Порівняння звичайної та lambda-функції:")
print(f"Звичайна функція 5*4: {multiply_func(5, 4)}")
print(f"Лямбда-функція 5*4: {multiply_lambda(5, 4)}")

# Приклад 3: Використання lambda як аргумента функції
coordinates = [(1, 5), (10, 2), (3, 8)]
# Сортуємо список за другою координатою (y)
sorted_coords = sorted(coordinates, key=lambda p: p[1])
print(f"\n2. Лямбда як ключ сортування: {sorted_coords}")
```
   1. Отримано наступні результати:
```
1. Порівняння звичайної та lambda-функції:

    Звичайна функція 5*4: 20
    Лямбда-функція 5*4: 20

2. Лямбда як ключ сортування: [(10, 2), (1, 5), (3, 8)]
```
